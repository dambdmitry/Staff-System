<Контракт по повторному использованию javascript скриптов>
Как использовать:
    1)selectInTableScript - узконаправлен на выбор строки и посыл нужных данных на форму (только для удаления)
        -пометить в таблицу id = "table"
        -строкам таблицы добавить класс "clickable-row"
        -отметить колонку строки, которую надо отправить на форму "forDelete"
        -в форме соответсвующему инпуту присвоить id = "propertyForDelete"
        -кнопке удалить' в форме присвоить id = "btnDelete" (Желательно еще сделать ee в html disabled)
    2)hideElementScript - контракта нет, работает только для добавления приказа.
